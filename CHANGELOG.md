# Changelog

All notable changes to this project will be documented in this file.

Format follows [Keep a Changelog](https://keepachangelog.com/).

## [1.0.0] - 2026-04-14

### Added
- Complete therapeutic framework with 5 modalities: CBT, DBT, ACT, Psychodynamic, and Eclectic
- 4 slash commands: `/session`, `/progress`, `/homework`, `/formulation`
- Automatic language detection — therapist speaks the client's language
- LLM Wiki pattern: persistent, cross-referenced therapeutic knowledge base
- Structured onboarding flow with modality education and goal setting
- Post-session wiki compilation: patterns, themes, insights, interventions, progress
- Homework system with active/completed tracking
- Crisis protocol with international hotline support
- Session pacing with soft/hard exchange limits
- Case formulation that evolves over sessions
- Quarterly wiki review protocol (~every 12 sessions)
- Setup script for global install via symlinks
